# portaDAC
Portable digital to analog converter
To view the PCB (Altium) Project, Altium Designer version 20.2.6 or later is required. Alternatively the latest version of the free Altium Designer viewer (Available at https://www.altium.com/viewer/) is reqired.
To view the SOLIDworks file, Solidworks 2019 or later is required. Alternatively the latest version of SOLIDWorks eDrawings can be used.
